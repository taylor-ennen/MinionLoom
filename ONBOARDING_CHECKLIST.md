# MinionLoom Onboarding Checklist

This checklist ensures all contributors and AI agents are properly set up to use MinionLoom's promptable commands and agent features.

---

## 1. Environment Setup
- [ ] Run `.github/minions/install_minions.ps1` to set up your environment
- [ ] (See the main [README.md](../../README.md) for initial project setup)

## 2. Prompt Path Configuration
- [ ] Confirm all prompt files are in `.github/minions/prompts/`
- [ ] Run `.github/minions/scripts/setup-vscode-prompt-path.ps1` to configure VS Code to include `.github/minions/prompts`
- [ ] Test `/research`, `/status`, `/continue` in your chat/CLI or Copilot

## 3. Read Documentation
- [ ] Review the main [README.md](../../README.md)
- [ ] Read the [AI_SETUP_GUIDE.md](AI_SETUP_GUIDE.md) for prompt integration

## 4. Troubleshooting
- [ ] If prompts do not appear, verify the prompt path and tool configuration
- [ ] Consult the setup guide for more help

---
This checklist should be used by all new contributors and AI agents to ensure a consistent and working setup.
