# /status

Use this command to:
- Report the status of a task, agent, or the system
- Summarize recent activity or errors
- Provide actionable next steps

## Usage
/status [task_id|agent|system]

## Example
/status system

---
This prompt is used by the Copilot CLI/chat and MinionLoom agents to provide status updates.
