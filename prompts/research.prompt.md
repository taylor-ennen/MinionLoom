# /research

Use this command to:
- Gather information on a topic
- Summarize documentation or code
- Provide references or links

## Usage
/research <topic or question>

## Example
/research How does the MinionLoom orchestrator manage task state?

---
This prompt is used by the Copilot CLI/chat and MinionLoom agents to perform research tasks efficiently.
